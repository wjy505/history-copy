# 需求规格说明

## 产品概述

历史粘贴板 — Windows 剪贴板历史管理软件。运行于系统托盘，自动记录用户复制（Ctrl+C）的所有文字和图片内容。用户可通过快捷键呼出历史面板，查看、搜索、置顶、删除已记录的内容，并双击卡片将内容粘贴到当前应用程序。

## 目标用户

- 经常需要查找之前复制过的内容的用户
- 需要保留短期（1-5天）复制历史的用户
- 偏好简洁直观界面的 Windows 用户

## 功能需求

### F1. 剪贴板监听
- 后台静默运行，自动捕获所有复制操作
- 支持格式：纯文本（CF_UNICODETEXT）、图片（CF_DIB/CF_BITMAP）
- 去重：相同内容短时间内多次复制只保留一条记录
- 去重算法：SHA256 哈希比对，300ms 防抖

### F2. 历史记录存储
- 存储位置：`%APPDATA%/ClipboardHistory/`
- 文字内容存储在 SQLite 数据库
- 图片以 PNG 格式存储在 `images/` 子目录
- 自动生成图片缩略图（200px 宽）

### F3. 历史面板
- 快捷键（默认 Alt+V）呼出面板
- 面板窗口置顶显示，失去焦点自动隐藏
- 卡片式排列，按时间降序（最新在上）
- 置顶卡片排在最前面，带有蓝色左边框标识
- 面板大小可调整，位置可记忆

### F4. 卡片交互
- 文字卡片：显示前 3 行预览 + 相对时间（如"3分钟前"）
- 图片卡片：显示缩略图 + 时间
- 悬停显示完整文字内容（tooltip）
- 双击卡片：将内容粘贴到前台窗口
- 置顶按钮：切换置顶状态
- 删除按钮：两次点击确认后删除

### F5. 搜索功能
- 搜索栏位于面板顶部
- 搜索范围：文字内容的全文搜索
- 实时搜索：输入后 300ms 防抖触发
- 搜索结果替换当前卡片列表显示
- 清空搜索框恢复完整列表

### F6. 粘贴功能
- 双击卡片后自动粘贴到之前活跃的窗口
- 粘贴流程：设置剪贴板 → 恢复前台窗口 → 模拟 Ctrl+V
- 如果前台窗口已失效，将内容复制到剪贴板并提示用户手动粘贴

### F7. 存储期限管理
- 可配置期限：1天 / 3天 / 5天
- 超过期限的非置顶项自动删除（含图片文件）
- 置顶项永久保留，不受期限影响
- 每小时执行一次清理检查

### F8. 系统托盘
- 托盘图标常驻
- 左键点击：显示/隐藏面板
- 右键菜单：
  - "显示历史面板" + 快捷键提示
  - "设置..."
  - 分隔线
  - "退出"

### F9. 设置对话框
- 存储期限选择（下拉：1/3/5 天）
- 快捷键设置（点击捕获按键）
- 图片最大宽度设置
- 开机启动选项
- 清空所有历史按钮（带确认）

## 非功能需求

### NF1. 性能
- 面板打开延迟 < 200ms
- 搜索响应 < 300ms
- 剪贴板捕获延迟 < 100ms
- 内存占用 < 100MB

### NF2. 可靠性
- 应用崩溃后剪贴板链自动恢复
- 数据库使用 WAL 模式防止写入损坏
- 启动时验证数据库完整性

### NF3. 兼容性
- 操作系统：Windows 10 / Windows 11
- 高 DPI 适配（100%-200% 缩放）
- 多显示器支持

### NF4. 可用性
- 单实例运行，重复启动激活已有窗口
- 配置和数据存储在用户目录，无需管理员权限
- 打包为单个 exe 文件，无需安装

## UI 设计约束

- 主色调：淡蓝色（#4A90D9）
- 设计风格：简洁、直观、扁平化
- 面板尺寸：默认 400×600px
- 字体：系统默认中文字体
- 所有文字使用中文
