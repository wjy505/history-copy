# UI 设计规范

## 色彩体系

| 用途 | 色值 | 说明 |
|---|---|---|
| 主色 | `#4A90D9` | 按钮、强调元素 |
| 主色深 | `#357ABD` | 悬停/按下状态 |
| 背景 | `#F0F4F8` | 面板背景 |
| 卡片背景 | `#FFFFFF` | 卡片底色 |
| 文字主色 | `#2C3E50` | 标题文字 |
| 文字次要 | `#7F8C8D` | 时间信息 |
| 强调色 | `#5DADE2` | 置顶边框、链接 |
| 危险色 | `#E74C3C` | 删除按钮 |
| 边框色 | `#D5DDE5` | 卡片边框 |

## 面板窗口

- 默认尺寸：400 × 600 像素
- 可拖拽调整大小（最小 300 × 400）
- 无标题栏（`Qt.FramelessWindowHint`）
- 置顶显示（`Qt.WindowStaysOnTopHint`）
- 失去焦点自动隐藏
- 出现位置：鼠标所在屏幕，靠近鼠标光标
- 圆角窗口：8px
- 阴影效果：`QGraphicsDropShadowEffect`

## 搜索栏

- 高度：36px
- 输入框圆角：18px（胶囊型）
- 左侧搜索图标（🔍），右侧清除按钮（✕）
- 占位文字："搜索剪贴板历史..."
- 背景：白色，边框 1px `#D5DDE5`
- 聚焦时边框变为主色 `#4A90D9`

## 文字卡片

- 高度：自适应（最小 60px，最大 120px）
- 内边距：12px
- 布局：左侧文字预览 + 右侧操作按钮
- 文字预览行数：最多 3 行，超出省略号
- 时间格式：
  - 1分钟内：「刚刚」
  - 1小时内：「X 分钟前」
  - 当天：「X 小时前」
  - 昨天：「昨天 HH:MM」
  - 更早：「YYYY-MM-DD HH:MM」
- 悬停效果：背景变 `#F8FAFB`，轻微阴影
- 置顶卡片：左侧 3px `#4A90D9` 色边框
- 操作按钮（默认半透明，悬停不透明）：
  - 📌 置顶按钮（14px，右上角）
  - ❌ 删除按钮（14px，置顶按钮右侧）

## 图片卡片

- 高度：120px
- 缩略图居中显示，宽度 200px，保持比例
- 缩略图圆角：4px
- 底部显示时间文字
- 操作按钮布局同文字卡片

## 系统托盘

- 图标：16×16 / 32×32 应用图标
- 提示文字：「历史粘贴板 - X 条记录」
- 右键菜单：
  - 显示历史面板 [Alt+V]
  - 设置...
  - ─────────
  - 退出

## 设置对话框

- 模态窗口：300 × 250 像素
- 标题：「设置」
- 表单项：
  - 存储期限：下拉选择（1天 / 3天 / 5天）
  - 快捷键：可点击文本框（点击后捕获按键）
  - 图片宽度限制：数字输入框
  - 开机启动：勾选框
  - [清空所有历史] 按钮（红色文字）
- 底部：确认 / 取消 按钮

## 动效

- 面板出现：200ms 淡入（`QPropertyAnimation` on `windowOpacity`）
- 面板隐藏：150ms 淡出
- 卡片插入：无动效（直接渲染）
- 按钮悬停：颜色平滑过渡 150ms

## 字体

- 系统默认字体（`QFont` 不指定 family）
- 正文大小：9pt
- 时间文字：8pt
- 搜索栏：9pt
